# Welcome

Luxa CSS is the minimal and super simple CSS framework for creating beautiful websites.

With just the right amount of everything, Luxa CSS is a great starting point for your project development.

## Key Features

### Dead-simple class names

The names of our classes are as simple as possible, and we have an even simpler system of modifiers.

### Just CSS

No dependencies, package managers, external files or JavaScript.

### Responsive everything

Fully adaptable and consistent spacing and typography across devices.

### Light or dark mode by default

Built already with two beautiful colour themes and activated automatically according to the user's preference.
